# 🛡 CyberShield Network Threat Monitor

## 📌 Project Overview
CyberShield Network Threat Monitor is a real-time cybersecurity monitoring dashboard developed using Python, Scapy, and Streamlit.

The project captures live network packets, analyzes protocols, detects suspicious traffic, and visualizes cybersecurity analytics through an interactive dashboard.

---

## 🚀 Features

- Real-time packet capture
- TCP, UDP, and ICMP protocol analysis
- Source & Destination IP tracking
- Suspicious traffic detection
- Threat level monitoring
- Interactive cybersecurity dashboard
- Protocol distribution analytics
- Packet statistics visualization

---

## 🛠 Technologies Used

- Python
- Scapy
- Streamlit
- Pandas
- Npcap
